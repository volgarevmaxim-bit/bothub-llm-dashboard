# BotHub LLM Dashboard 📊

**Анализ затрат LLM-моделей** из выгрузки BotHub.  
Работает локально в браузере — **без сервера, без установки**.

## Как использовать

### 1. Скачать данные

1. Откройте [BotHub → Настройки → Транзакции](https://bothub.ru/ru/profile/settings/transactions)
2. Нажмите кнопку **«Скачать XLSX»** (выгрузка всех транзакций)

### 2. Загрузить в дашборд

1. Скачайте дашборд одной кнопкой:  
   **[⬇ bothub-llm-dashboard.zip](https://github.com/volgarevmaxim-bit/bothub-llm-dashboard/releases/download/v1.0/bothub-llm-dashboard.zip)**  
   *(в архиве только один файл `bothub-llm-dashboard.html`)*
2. Распакуйте архив, откройте **`bothub-llm-dashboard.html`** в браузере
3. Перетащите скачанный XLSX-файл в зону загрузки  
   — или нажмите «Выбрать файл»
4. Дашборд обновится автоматически

## Что показывает

| Блок | Описание |
|------|----------|
| **KPI** | Всего caps, стоимость в рублях, количество запросов |
| **Затраты по модели** | Рейтинг моделей по стоимости (горизонтальный барчарт) |
| **Доля по типу вызова** | Круговая диаграмма — распределение по семействам моделей (Claude, DeepSeek, GLM и т.д.) |
| **Активность по времени** | График caps/час для всех моделей |
| **Таблица детализации** | Транзакции, caps, стоимость, токены, **кэш хит** с цветовой индикацией |

## Цвета кэш-хита

- 🟢 **>25%** — отлично
- 🟡 **10–25%** — повод для оптимизации
- 🔴 **<10% или нет кэша** — плохо

## Формат входных данных

Ожидается XLSX-выгрузка из BotHub со следующими колонками:

| Колонка | Описание |
|---------|----------|
| `Дата (UTC+0)` | Дата и время запроса |
| `CAPS` | Потрачено caps |
| `model_id` | Имя модели |
| `Входных токенов` | Input токены |
| `Выходных токенов` | Output токены |
| `Кэшированных токенов` | Cached токены |
| `Провайдер` | Провайдер |

- Чистый HTML/CSS/JS — никаких зависимостей, бэкенда или сборок
